# Aeris UI Support

Aeris UI is currently alpha software. Prereleases receive best-effort support and are not covered
by a production stability commitment.

- Use the [documentation](https://aeris-ui.dev) and component API pages for usage guidance.
- Use [GitHub Issues](https://github.com/Aeris-UI/AerisUI/issues) for reproducible bugs and focused
  feature requests.
- Use GitHub Discussions for broader implementation questions when Discussions are enabled.
- Use [GitHub Security Advisories](https://github.com/Aeris-UI/AerisUI/security/advisories/new) for
  suspected vulnerabilities.

Do not include credentials, private application code, personal data, or undisclosed security
details in public issues or discussions.
