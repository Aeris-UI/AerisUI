# Documentation asset provenance

The files in this directory are used only by the Aeris UI documentation application. They are not
part of the published `@aeris-ui/core` package.

## Aeris UI brand assets

The following original Aeris UI brand assets were supplied by the project maintainer. They are not
third-party Pexels content:

- `logo.png`
- `favicon.ico`
- `favicon-16x16.png`
- `favicon-32x32.png`
- `apple-touch-icon.png`
- `android-chrome-192x192.png`
- `android-chrome-512x512.png`

The favicon and application-icon files are derivatives of the Aeris UI logo. They are included to
identify the Aeris UI project and documentation.
