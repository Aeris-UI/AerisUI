Accessibility

  • The list uses listbox and option semantics with configurable accessible naming.
  • Multiple mode sets aria-multiselectable and each item exposes aria-selected.
  • Reorder controls are native buttons with configurable labels.
  • Disabled state removes the listbox from the tab order and disables all item and reorder controls.

Keyboard support

KeyFunction
TabMoves focus through reorder controls and list items.
Enter / SpaceSelects or clears an item button.
Alt + ArrowUpMoves selected items up when the listbox has focus.
Alt + ArrowDownMoves selected items down when the listbox has focus.
Alt + HomeMoves selected items to the top.
Alt + EndMoves selected items to the bottom.
